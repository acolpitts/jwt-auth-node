# JWT Authentication API
JWT Authentication API with passport and mongodb.

#####Getting Started
1. Clone this repo
1. Install dependencies
1. Run server
1. *Do your thing*

```
> npm install
> npm run start
```

#####Depoly to Heroku
```
> heroky config:set
```
